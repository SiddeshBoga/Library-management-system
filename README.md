# Library Management System (C)

This project is a **console-based Library Management System** developed
using the C programming language. It was created to demonstrate strong
foundations in C programming, file handling, and structured programming
concepts.

The system allows users to manage book records efficiently using a
menu-driven interface and persistent storage through files.

---

## Project Overview
The Library Management System is designed to perform basic library
operations such as adding, viewing, and searching for books. The system
stores data in binary files, ensuring that records are preserved between
program executions.

This project focuses on:
- Understanding file handling in C
- Using structures for data organization
- Implementing menu-driven applications
- Handling user input and file I/O safely

---

## Features
- Add new books to the library
- Display all available books
- Search for books by ID or name
- Persistent data storage using binary files
- Menu-driven user interface
- Basic error handling for file operations

---

## Technologies Used
- Programming Language: C
- Concepts:  
  - Structures  
  - File Handling (Binary Files)  
  - Conditional Statements  
  - Loops and Functions  

---

## How It Works
1. The program displays a menu with available options.
2. User selects an operation (add, view, search, exit).
3. Book data is stored and retrieved from a binary file.
4. Records remain available even after restarting the program.
